# NeoGram - Modern Messenger

Современный мессенджер с реальным обменом сообщениями, красивым интерфейсом и поиском пользователей.

## Возможности

✅ **Красивый современный интерфейс**
- Темная тема с градиентами
- Анимации и плавные переходы
- Адаптивный дизайн для всех устройств

✅ **Полная система чатов**
- Переключение между чатами
- Индикаторы непрочитанных сообщений
- Время последнего сообщения
- Статусы пользователей (online/offline)

✅ **Поиск пользователей**
- Поиск по username (например @monkey)
- Быстрый доступ к популярным пользователям
- Информация о пользователе перед началом чата

✅ **База данных**
- Хранение пользователей, чатов и сообщений
- Статусы пользователей в реальном времени
- История переписок

✅ **WebSocket соединение**
- Мгновенная доставка сообщений
- Индикатор "печатает..."
- Статусы прочтения сообщений
- Обновление статусов в реальном времени

✅ **Система аутентификации**
- Регистрация по номеру телефона
- Подтверждение по SMS коду
- Создание профиля с username
